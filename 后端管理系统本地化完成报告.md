## 技术细节

### 翻译系统架构
- 使用 React Context API 提供全局翻译功能
- 通过 `useI18n()` hook 访问翻译函数 `t()`
- 支持嵌套键路径（如 `admin.monitor.title`）
- 自动回退到中文翻译（如果当前语言缺失）

### 语言切换
用户可以在设置中切换语言，所有管理界面会立即响应：
- 英文 (en)
- 中文 (zh)

## 验证方法

1. 启动应用：`npm run dev`
2. 登录管理后台
3. 在设置中切换语言
